Hand-typed in minimal HTML/CSS in Notepad, stored in Dropbox, transferred by FileZilla through FTP, hosted by WPX. Set in Palatino Linotype.

Colors: #f6e6cf (background), #503832 (text/borders,) and #ffd6a5 (highlighted text.)

Source code available here.

Accessibility features: semantic HTML and a skip/back to top link.

Privacy: no analytics, scripts, tracking, cookies, ever.

Designed for pleasurable reading, inspired these design ethe:

  • Against digital pollution.
  • A personal website is a bit like a quilt.
  • Built to last.
  • Cool URIs don't change.

Checklists

Publishing A New Page

  • Open 'template.html' in Notepad.
  • Copy the HTML template provided into a new Notepad document.
  • Save and name the file in Dropbox.
  • Write the contents in Markdown and convert to HTML.
  • Update the title tag.
  • Fill in the meta name="description".
  • Update meta property="og:title".
  • Update meta property="og:url".
  • Update publication date in meta name="date"> tag and in p class="postmeta".
  • Add the slug to the 'skip' and 'return to top' link.
  • Update 'Copy & Share' link.
  • Calculate and update word count and estimated reading time in 'Statistics.'
  • Open the HTML file in a web browser to see how it looks and test the links.
  • Proofread and fix contents.
  • Update the changelog and the word count.
  • Update the sitemap and the word count.
  • Update the homepage word count and updated date.
  • Update the sitemap.xml file.
  • Update the rss.xml file if applicable.
  • Upload to WPX using FileZilla.
  • Check it's live on the site and for errors, fix as needed.
  • It gets posted as a Gist on Github automatically.
  • Go to the GitHub repository.
  • Upload the new file and commit the changes.
  • Update changelog.html, sitemap.html, sitemap.xml and rss.xml files on GitHub.
